# Weather Metrics Dashboard

A web dashboard for exploring 20 years of historical Australian weather data, built with a Java back end and a vanilla HTML/CSS front end backed by a SQL database.

## What it does

- Displays historical weather metrics — rainfall, temperature (min/max), humidity, and more — spanning two decades of records.
- Lets users filter and explore data by **state** and **weather station**, surfacing regional climate trends over time.
- Queries a relational database directly with SQL to aggregate and filter large historical datasets efficiently.
- Renders results through a clean, responsive interface built with plain HTML and CSS — no front-end framework dependencies.

## Skills demonstrated

- SQL query design and data filtering
- Server-side rendering with Java
- Responsive front-end design without a framework
- Working with real-world, multi-decade datasets

## Tech stack

Java, HTML, CSS, SQL
